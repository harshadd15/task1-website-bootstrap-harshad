DOCUMENTATION :
Header (Navigation & Branding)
Shows the HotelFinder logo and menu links (Find Hotels, Hotels, Contact) so users can
easily navigate the site.
Hero Section (First Impression)
Introduces the website and encourages users to search for hotels.
Search Box (Quick Search)
Lets users search hotels by location, date, and price easily.
Hotel Cards (Hotel Display)
Shows different hotel types like Budget, Luxury, and Resorts in card format.
Modal (Hotel Details)
Opens a popup with hotel details, price, facilities, and a Book Now button.
Carousel (Images Slider)
Displays hotel images in a sliding format to make the site attractive.
Contact Section (User Messages)
Allows users to send questions or messages.
Footer (Ending Section)
Shows copyright information at the bottom of the page.
Offcanvas (Mobile Menu)
Provides a hidden menu for mobile users to save screen space.
HTML Tags Used (Simple)
• <html> – Main document
• <head> – Page settings and styles
• <title> – Website title
• <meta> – Responsive and charset info
• <link> – External CSS (Bootstrap, Fonts)
• <style> – Custom CSS
• <body> – Website content
• <nav> – Navigation
• <section> – Page sections
• <div> – Layout containers
• <h1>–<h6> – Headings
• <p> – Text
• <a> – Links
• <img> – Images
• <ul>, <li> – Lists
• <button> – Buttons
• <form>, <input> – Forms
Custom Features
• Header – Custom design using grid, colors, and shadow
Buttons – Rounded, colorful, and hover effects
Carousel – Smooth sliding images with fixed height
Offcanvas – Stylish mobile menu with smooth animation
Grid System – Responsive layout for cards and search box
